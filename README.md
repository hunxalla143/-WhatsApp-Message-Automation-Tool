# WhatsApp Message Automation Tool

Send WhatsApp Messages

Bulk Messaging

CSV/JSON Contact Support

Personalized Templates

Scheduling System

Message Logging

Duplicate Prevention

Media Sending

pip install pywhatkit pandas schedule colorama
